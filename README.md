# Riverhaus · Lumen Studio

Sitio web de Riverhaus — Next.js 16 + Sanity CMS + Tailwind CSS v4.

## 🌐 Ver sitio

**https://cool-chatterjee-9dea31-vicentemattas-projects.vercel.app**

## Stack

- **Next.js 16** (App Router, Turbopack)
- **Sanity CMS** (next-sanity 12, GROQ, Studio embebido en `/studio`)
- **Tailwind CSS v4** + **Framer Motion**
- **Cargo CDN** para media (videos e imágenes)

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).
